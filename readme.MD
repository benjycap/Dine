# Running locally:

## Pre-reqs:
[Node 11.10.0](https://nodejs.org/en/download/package-manager/)  
[mongodb](https://docs.mongodb.com/v3.2/administration/install-community/)

## Run:
- Start local mongodb env
```
$ npm install
$ npm run build
$ npm start
```
Go to http://localhost:3000
